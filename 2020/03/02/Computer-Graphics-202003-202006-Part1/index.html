Lecture1 Graphics adapters(图形适配器), colors and 2D drawings

Graphics adapters

  • Vector graphic adapters (ancient history)

  • Raster graphic adapters(光栅)

    • Have a special memory, called Video Memory (VRAM).

  • Accelerated graphic adapters

    • special types of raster graphic adapters that have much more memory than the one required to store a single screenshot.

2D graphics primitives

  1. 2D Graphics Primitives - coordinates system
  2. Pixel coordinates(hardware coordinates)
    • Note that the coordinate system is left-handed, since it has the y-axis in the opposite direction compared to the classical one.
    • $ 0 <= x <= sw -1$ $0 <= y <= sh -1 $
  3. Clipping
  4. Normalized Screen Coordinates (102:00 未解决的问题 cut ??)
    • Goal: Applications want to display the same image regardless of the resolution and the size of the screen. They also want to exploit all the features of the display at their best.
    • It’s a Cartesian coordinate system, where x and y values range between two canonical values (generally between -1 and 1, but other standards such as 0 and 1 also exists).
